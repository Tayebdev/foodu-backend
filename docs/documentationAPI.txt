CLASS: User
FILENAME: User.txt
================================================================================
Class Name: User
Description: Represents the base user entity in the system, managing authentication and profile details.
Attributes:
  - userId: String (PK) - Unique identifier for the user.
  - name: String - Full name of the user.
  - email: String (Unique) - Email address for login and notifications.
  - phone: String (Unique) - Contact phone number.
  - passwordHash: String - Encrypted password.
  - role: Enum{Admin, Client, RestaurantOwner, Driver} - Role determining permissions.
  - picture: String - URL to profile picture.
  - verified: Boolean - Information verification status.
  - isActive: Boolean - Account active status.
  - isBanned: Boolean - Account ban status.
  - verifyCode: String - Code used for email/phone verification.
  - createdAt: Date - Account creation timestamp.
  - updatedAt: Date - Last update timestamp.
  - lastLogin: Date - Timestamp of last login.

Methods / Operations:
  - register(email, password, role)
    Route: POST /auth/register
    Description: Registers a new user.
    Parameters: email (String), password (String), role (Enum)
    Returns: User - Created user object.
    Request JSON: { "email": "user@example.com", "password": "securePass123", "name": "John Doe", "role": "Client" }
    
  - login(email, password)
    Route: POST /auth/login
    Description: Authenticates a user and returns a token.
    Parameters: email (String), password (String)
    Returns: String - JWT Token.
    Request JSON: { "email": "user@example.com", "password": "securePass123" }
    
  - logout()
    Route: POST /auth/logout
    Description: Logs out the current user.
    Returns: void

  - updateProfile(data)
    Route: PUT /users/profile
    Description: Updates user profile information.
    Parameters: data (JSON) - Fields to update.
    Returns: User - Updated user object.

  - changePassword(oldPassword, newPassword)
    Route: PATCH /users/change-password
    Description: Changes the user's password.
    Parameters: oldPassword (String), newPassword (String)
    Returns: Boolean - Success status.
    
  - resetPassword(token, newPassword)
    Route: POST /auth/reset-password
    Description: Resets password using a recovery token.
    Parameters: token (String), newPassword (String)
    Returns: Boolean

  - verifyEmail(token)
    Route: GET /auth/verify-email
    Description: Verifies user email address.
    Parameters: token (String)
    Returns: Boolean

  - deleteAccount()
    Route: DELETE /users/me
    Description: Deletes the current user's account.
    Returns: void

  - getById(userId)
    Route: GET /users/{userId}
    Description: Retrieves user details by ID.
    Parameters: userId (String)
    Returns: User

  - ban(reason)
    Route: POST /users/{userId}/ban
    Description: Bans a user (Admin only).
    Parameters: reason (String)
    Returns: void

  - unban()
    Route: POST /users/{userId}/unban
    Description: Unbans a user (Admin only).
    Returns: void
